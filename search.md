The [[Agora]] offers several ways to find what you are looking for.

## Search Bar
The search bar at the top of the page allows you to:
*   **Jump to a Node:** If you type the exact name of a node (e.g., `protopia`), you will go there directly.
*   **Fuzzy Search:** If the node doesn't exist, we'll try to find the closest match.
*   **Full Text:** We are working on better full-text search capabilities.

## URL Navigation
You can navigate directly using your browser's address bar:
*   `anagora.org/node-name`: Go to a node. Spaces should be replaced with `+` or `-` (e.g., `anagora.org/social+contract`).
*   `anagora.org/go/shortcut`: Use [[go links]] (if configured).

## Wikilinks
The primary way to navigate is by following [[wikilinks]].
*   Links in `[[double brackets]]` connect concepts.
*   If a link is **blue**, the node exists.
*   If a link is **red** (or distinct), the node doesn't exist yet—clicking it invites you to create it!
